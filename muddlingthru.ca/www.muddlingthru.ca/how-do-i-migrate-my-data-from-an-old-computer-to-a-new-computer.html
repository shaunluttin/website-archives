How do I migrate my data from an old computer to a new computer (and other questions)?

Wednesday, September 11, 2013 5:41:00 PM

How do we migrate my data from an old computer to a new computer?

Our Progress

  1. Install Windows Easy Transfer on Windows XP. Done.
  2. Run Windows Easy Transfer on Windows XP. Partly done. It needs more HDD space to operate.
  3. Manually transfer the contents of Adminstrator/MyPictures to free up space on the old computer. In progress. 
  4. Run easy transfer again on the old computer.

What does my client want to transfer?

  • Most but not all of his photos. This is doable.
  • Maybe some documents.
  • Maybe some browser bookmarks.
  • Most if not all of his emails. He uses Firefox and the Gmail cloud mail system to access email. That means we do NOT have to worry about transfer any of his emails, because they are in the cloud not on his old computer. This could have been more complicate if he was using MS Outlook or Windows Mail for email access. Yah for the cloud!
  • The programs to browse the web, listen to music, and view photos. This is not necessary because Windows 8 will have its own built in software to browse the web, listen to music, and view photos.
  • Result: we only have to transfer photos. Easy. Kinda.

New Computer

  • Windows 8 Operating System
  • 6 GB RAM
  • 64 bit architecture
  • ADS A4-4300M APU with Radeon HD Graphics 2.50 GHz

Old Computer

  • Windows XP Professional SP 3 Operating System
  • 512 MB RAM
  • 32 bit architecture
  • Intel Pentium Professor 1.60 GHz 988 MHz

Steps

  1. Install Windows Easy Transfer on the Windows XP computer.*
  2. Sign in to both PCs with an admin account.
  3. Choose a transfer option. I recommend a Flash Drive (aka a memory stick).
  4. Run anti-virus before transfer on the old PC.
  5. Run anti-virus after transfer on the new PC.

Other Questions

How do I manage my myriad usernames and passwords?

I recommend using LastPass. We can talk more about that later. https://lastpass.com/

What anti-virus software should I use on Windows 8?

I recommend using Windows Defender not another anti-virus. Avoid Norton, avoid Avast, avoid AVG. They will muck up your computer and badger you for money.

Progress

  • Uninstall Norton from Windows using the Control Panel > Programs and Features. Done.
  • Also uninstall Norton Online Backup. Done.
  • Also uninstall Norton Security Dashboard. Done.
  • Restart the computer. Done.
  • Enable Windows Defender as the alternative. Done. When we unistall Norton, Windows Defender enables itself.

Homework

Consider purchasing a USB Flash drive of around 32 GB.

Use the browser (Firefox, Internet Explorer, Safari, Chrome, etc) in Windows 8 to access your gmail account. Here is a list of popular ones: http://www.w3schools.com/browsers/default.asp

Unistall Norton and install Windows Defender as the antivirus tool. Done.

Uninstall one program that you are not going to use and that you are *sure* the computer doesn't need. When in doubt, uninstall it. You can often tell whether the program is important by looking at its manufacturer (e.g. if Microsoft is the manufacturer, think twice about unistalling it).

  1. Open the Charms Bar.
  2. Search Control Panel.
  3. Open Control Panel.
  4. Open Programs and Features.
  5. Click a program name that you do not want.
  6. Click uninstall.
  7. Follow the prompts.
  8. After uninstall, manually delete the shortcut icon from the desktop.
