HowTo: Update and remove software on my Apple laptop

Friday, January 10, 2014 6:56:00 PM

Topics

  • Update Browser
  • Update Adobe
  • Remove Google Earth and install it again because it isn't working right now. Done. We dragged all the Google Earth related icons into the Trash to uninstall it. We then downloaded Google Earth by using the Safari browser. Finally, we dragged Google Earth into the harddrive as per the instructions. Google Earth, though, did not work after all this. More investigation revealed that it was actually version 7.0; this version does not work with Mac OS X 10.4 and below. So, we searched for and downloaded Google Earth 6.1; this version is supposed to have support for OS X 10.4. This didn't work either. It turns out that the Google documentation was incorrect and that from the Apple documentation we need Google Earth 5.2.1.1588 for OS X 10.4 (https://www.macupdate.com/app/mac/20124/google-earth/). Having installing this version, we get farther but after showing the Google Earth screen the application unexpectedly shuts down. It almost installed! 

Summary

Google Earth is basically broken for the Mac OS X 10.4. We succeeded at using the Google Earth Web Plugin. Here is how to access it. 

  1. Open Safari. 
  2. Go to maps.google.com
  3. Wait for it to load.
  4. Hover on the Satelight icon in the upper right. 
  5. The Earth icon will pop up beside it. 
  6. Click the Earth icon. 
  7. Google Earth will load.
