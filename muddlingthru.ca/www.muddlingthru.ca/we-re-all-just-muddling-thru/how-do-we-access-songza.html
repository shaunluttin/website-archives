How do we access Songza without it crashing?

Friday, February 28, 2014 7:31:00 PM

Make Songza Run

Songza is a very modern website. As a result, it requires an up-to-date web browser. So, we need to update the web browser (e.g. Firefox). In fact, update Firefox as often as you can.

  1. Open Firefox.
  2. Go to Help. 
  3. Read whether "Firefox is up to date" or not.

Also, avoid runnin too many programs at once, because you will max out your computer capabilities (RAM, CPU), and Songza will complain by saying, "A long running script is doing something." That just means, an instruction that you've given the computer is bogging down.

Speed up my computer

  1. Start > Control Panel > System and Security > Administrative Tools > System Configuration > Startup > Uncheck any programs that you do NOT want to run on startup > Click apply. Done.
  2. Disable visual features. Done.
  3. There are lots of other tweaks too.
