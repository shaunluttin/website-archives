Running WordPress on Microsoft Azure, Notes

Tags: wordpress, azure

Wednesday, September 9, 2015 3:49:57 AM

Option #1 Azure Websites with ClearDB MySQL

  • The ClearDB is expensive even for modest workloads.
    • Mercury, not production ready - free
    • Titan, small Wordpress and home-brewed apps - $42 / year
    • Venus, personal Wordpress blogs or light staging application - $120 / year
    • https://www.cleardb.com/store/azure

Option #2 Azure Website with Azure VM MySQL

  • More complicated
  • Maybe less expensive
  • Perhaps start with Mercury. Then switch to this later by:
    • first restoring a backup from the ClearDB to the MySQL running on the Azure VM and
    • second modifying the WordPress configuration.
