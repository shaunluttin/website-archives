The basics of selecting and changing mutliple values in Excel.

Wednesday, September 18, 2013 5:47:00 PM

Select multiple adjascent cells

  1. Select the first cell.
  2. Hold down SHIFT.
  3. Select the last cell.
  4. Release SHIFT.
  5. All adjascent cells will now be selected.

Select mutliple non-adjascent cells

  1. Select the first cell.
  2. Hold down CTRL.
  3. Selet the second cell.
  4. Release CTRL.
  5. Repeat steps 2 through 5 until you've select all target cells.

Copy and paste

  1. Select the source cell(s).
  2. Play the CTRL + C chord (i.e. hold CTRL, then press C).
  3. Release CTRL + C.
  4. This copies the contents of the cell(s).
  5. The mouse is now free to select the destination cells.
  6. Select the destination cell(s).
  7. Play the CTRL + V chord.
  8. Release CTRL + V.
  9. This pastes the source into the destination.

Copy and paste notes

  • You can copy and paste from a single source cell into multiple destination cells.
  • You cannot easily copy and paste from multiple source cells into a single destination cell.

Find and replace

  • Play the CTRL + F chord.
  • The Find and Replace dialob box will appear.
  • You can use this to find all instances of a specific string.
  • You can also use this to replace all instance of a specific string with another one.
  • Combine this with the copy and paste functions and you're set.
