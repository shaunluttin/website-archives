Over Simplified WPF Layout

Thursday, April 10, 2014 3:03:00 PM

DockPanel

Defines an area where you can arrange child elements either horizontally or vertically, relative to each other.

Properties

Grid

Defines a flexible grid area that consists of columns and rows.

Properies

  • Margin Gets or sets the outer margin of an element. 
  • ColumnDefinitions Gets a ColumnDefinitionCollection defined on this instance of Grid.
  • RowDefinitions Gets a RowDefinitionCollection defined on this instance of Grid.

ColumnDefinition & RowDefinition

  • Defines column-specific or row-specific properties that apply to Grid elements.
  • Width & Height Take a GridLength value.

GridLength

Represents the length of elements that explicitly support Star unit types.

Values

  • doubleLength a Double value that specifies a device-independent units (1/96th inch per unit) measure
  • autoValue That include Auto or Star (*) as in GridUnitType
  • qualifiedDouble same as doubleLength but with specific units

GridUnitType

Auto resizes the column to fit its content

Pixel the value is expressed as a pixel

Star 

  • The value is expressed as a weighted proportion of available space.
  • star values are expressed as * or 2*.
  • In the first case, the row or column would receive one times the available space;
  • in the second case, the row or column would receive two times the available space, and so on.
