Very Useful Universal Keyboard Shortcuts for Productivity in Text Documents

Friday, January 3, 2014 6:14:00 PM

The following keyboard shortcuts work in most operating system and most program. I.e. They will work in Apple, Windows, Android, and most of their programs.

Correct Mistakes

  • ctrl+z Undo the last action
  • ctrl+y Redo (aka undo the last undo)
  • You can press these repeatedly to go further back or forward in time.
  • The segments of undo and redo are hard to predict.

Move Text Around

  • ctrl+x cut everything that is highlighted
  • ctrl+c copy everything that is highlighted
  • ctrl+v paste at the cursor location

Change the Cursor Location

  • arrow keys. go back one character, go forward one character, go up one line, go down one line.
  • home. go to the start of the line.
  • end. go to the end of the line.

Select Text

  • hold shift. this will highlight the characters over which you are about to traverse with the cursor location keys.
  • ctrl+a select all text on a page (hit any arrow key to unselect all)

Change Text Styles

  • ctrl+b bold everything that is highlighted
  • trl+i italicize everything that is highlighted

Other

  • ctrl+s save your current document
  • ctrl+f find text in your document
  • ctrl+p print the document
