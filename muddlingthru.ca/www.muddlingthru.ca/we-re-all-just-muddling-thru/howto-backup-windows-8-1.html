HowTo: Backup Windows 8.1

Thursday, March 20, 2014 6:57:00 PM

There are two main options for a Windows 8.1 backup:

  • File History.
  • System Image.

We will look at each of them. Let's start with File History, because it's the most important and useful.

File History

File History creates a backup of your personal data. It creates snapshots of your files whenever they change, and it stores incremental versions of those files on an external hard drive. You can then flip through the history of your files and restore any one of them to how it was at any particular point in time. This is a terrific way of both backing up and preserving-the-evolution of your personal files.

How do I turn on file history?

  1. Plug in an external hard disk drive. You can buy one from Brendan at Salt Spring Computer.
  2. From within the Windows 8.1 Start Screen, type "File History"
  3. In the search results, click "File History settings"
  4. The File History Settings pane will open.
  5. Turn it "On"
  6. Point it at your external hard disk drive.
  7. You will see a notice that says, "File History is backing up your files."
  8. Good. You're done.
  9. File History will now automatically backup your files, on a regular basis (even for a while when your external drive is not connected).

What does file history backup?

File history backs up everything in the following locations:

  • Desktop Folder
  • Favorites Folder
  • Contacts Folder
  • The Default Libraries, which include these Libraries:
    • Music Library
    • Videos Library
    • Documents Library
    • Pictures Library
    • Downloads Library
  • Custom Libraries

Note: Backing up Custom Libraries is possible but it can be buggy. That is, I have made it work but I cannot yet guarentee its reliability.

How do I restore a file from File History?

To restore your files, first open File History. Here are two methods:

Method A
  1. From within the Windows 8 Start Screen, type "Restore"
  2. Click "Restore your files with File History."
  3. File History will open at its root location (i.e. at a birds-eye-view of its contents).
Method B
  1. From within the Windows File Explorer, click on any file or folder that is part of File History (e.g. Documents).
  2. In the File Explorer ribbon, click the Home tab, and in its Open secion, click the History button (it looks like a folder with a clock.)
  3. File history will open at that location.

Once you have File History open, use its navigation, that is located at the bottom of the window, and which includes a big, green button between big forward and backward arrows.

Click the backward arrow to view earlier versions of files and folders. Click the forward arrow to return to the present. The big green button means, "restore to its original location." So, once you find something that you'd like to restore, select the file or folder, and then press the big green button. That will restore the item to its original location.

Right clicking a file or folder in File History displays further options. You can preview content or restore an item to a particular location instead of to its original location. Restoring something to a particular location comes in handy if you want to restore something while also using its current version.

How often does File History backup my files?

  • By default, File History saves a copy of your files every hour.
  • You can see this for yourself (and change this) in the Advanced Settings.

For how long does File History keep my files?

  • By default, File History keeps saved versions of your files forever.
  • You can see this for yourself (and change this) in the Advanced Settings.

If it saves my files every hour and keeps them forever, doesn't File History use an giganticly enormous amount of memory?

  • No. Every time File History saves a file, it only saves what changed, it doesn't re-save the entire file.
  • An anology is that, if you grew an inch, we would just raise the mark on the wall up one inch.

What does File History do when my drive is not connected?

  • When your external drive is not connected, file history uses your built in hard drive to backup files.
  • When you reconnect your external drive, File History transfers this local cache to the external drive.
  • So, File History works both when your external drive is connect and for a while when it isn't connected.
  • You can set the amount of your built in drive that File History uses in Advanced Settings (it's called "Size of offline cache.")

How do I open Advanced Settings?

  1. From within the Windows 8 Start Screen, type "File History"
  2. Click the "File History" search result.
  3. The File History control panel will open.
  4. Click on the Advanced settings link (on the left side).
  5. Here you can view and/or set how often File History will save copies, how much of your build-in hard drive it uses, and for how long it keeps your backed-up files.

What happens if I switch the device that I plug into my designated USB drive?

  • If you switch the device that you plug into your USB Drive,
  • Windows will assign a new drive letter to the device.
  • So, you will have to point File History at the new drive letter.

Where does file history store itself?

This is for geeks only. View the backup here: "G:\FileHistory", where G:\ is your external drive. Most users will not want to view their files this way. Most uses also don't need to create a System Image, but here is an explaination anyway.

Create a System Image

File History is good for backing up and restoring your personal data as needed; on the other hand, a System Image is good for backing up your entire computer (everything) and restoring it wholesale when your hard drive crashes (and your hard drive will crash). It's probably good enough to use just File History, but sometimes you might want a System Image. For instance, if you're about to do something that might destroy your hard drive, such as updating its firmware, then first create a system image.

  1. Plug in an external hard disk drive.
  2. From the start screen, type "File history" (yup, we go through File History).
  3. Click the "File History" result (not the "File History Settings" result).
  4. The File History Control Panel will open.
  5. In the bottom left, click System Image Backup link.
  6. The Create a system image window will open.
  7. A progress bar will also run for a while.
  8. Then the "Create a system image" window will show asking where you want to save your backup.
  9. Choose the external hard disk drive that you connected in step one of these instructions (you could also use some DVDs).
  10. Click next.
  11. Choose the drives that you want to include in the backup. I just accept the defaults.
  12. Click next.
  13. Read the warnings and then click Start the backup.
  14. The backup will take somewhere are 15 minutes to an hour.
