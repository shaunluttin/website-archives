HowTo: Do favorites...

Friday, March 14, 2014 6:16:00 PM

Favorites = Bookmarks = An item that you can open quickly once your browser is open. 

Shortcut = An item that you can open before your browser is open.

Browser = E.g. Firefox.

Tasks

Show the bookmark toolbar by default in Firefox. Done. View > Toolbars > Bookmarks Toolbar > Make sure there is a tick beside it.

Add a bookmark to the bookmarks toolbar in Firefox. Done.

  1. Go to the site that you want to bookmark.
  2. Double cllick the star on the right side of the address bar.
  3. The bookmarks context menu will open.
  4. Name: Give the bookmark a name.
  5. Folder: Choose the Bookmarks Toolbar folder.
  6. Click Done.

Questions

What, exactly, are the causes of the "There is a long running script..." error web browsing websites?

  1. One cause is that a local script is taking a long time to complete it's tasks. 
  2. Another cause is that a script is GETing something from the Internet but with a very slow or non-existant connection.
