Using Windows Live Writer with Orchard CMS

Tags: orchard-cms

Monday, June 16, 2014 8:00:24 PM

From the Orchard UI

  1. Turn on three modules:
    1. Blogs
    2. Remote Blog Publishing
    3. XMLRpc
  2. Create a new Blog. Note its Permalink in the blog settings.

From the Live Writer UI

  1. Add a blog in Windows Live Writer
  2. Choose “Other Services”
  3. Configure as follows:
    1. “Web Address of Your Blog,”, put the Permalink to your blog (E.g. http://www.muddlingthru.ca/we-re-all-just-muddling-thru)
    2. Username
    3. Password
  4. Live Writer, at this point, should just work.
