Office 365 Admin - Brain Dump

Tags: office-365, O365

Thursday, October 8, 2015 11:03:00 PM

Users

  • Admin > USERS > Active Users
  • This is where we add users and assign them licenses.

Contacts / People Address Books

  • Admin > CONTACTS lets us add contacts that appear in each user's Global Address List (GAL) but that only the O365 admin can edit. This is a good way to create shared contacts that only admins can edit. We can import contacts into the GAL using PowerShell 0365 and a CSV file.
  • Alternatively, using a Company Shared Contacts Folder provides greater control over who can edit. Configure this within Admin > ADMIN > Exchange > public folders. The benefit is that we can use this to allow non-admins to add/edit/delete contacts. The major downside is that public folders are only accessible via the Outlook Desktop App NOT via Outlook Web Access (OWA) nor mobile devices.

Email

  • Give a user email by assigning them a license (see Users)

Domains

  • Manage custom domain names from Admin > DOMAINS.
