Transfer music from a CD to Windows Phone 8 using Windows 8.1

Tags: windows-8.1, windows-phone, windows-8, music, CD

Sunday, October 19, 2014 3:44:03 PM

Steps

CD –> Windows Media Player –> Windows 8.1 PC –> Windows Phone App –> Windows Phone

  1. Rip the CD to your Windows 8.1 PC using Window Media Player.
  2. Transfer the music from your Windows 8.1 PC to your Window Phone using the Windows Phone App.
  3. Find the music on your phone.

Rip from CD to PC.

  1. Insert the CD (don’t do anything from Autoplay).
  2. Open Windows Media Player.
  3. Click “Rip Settings”
  4. Choose “Rip <CD Name>”

http://windows.microsoft.com/en-ca/windows-8/rip-burn

Transfer from PC to Phone.

  1. Install the Windows Phone App onto the PC.
  2. Plug your phone into the PC (make sure it’s charged first).
  3. Choose “Add to phone”.
  4. “Add music”.

Find the Music on the Phone.

  1. Swipe left to view all apps.
  2. Choose “Music + Videos”
  3. Under “Collection” choose “Music”
  4. If you don’t see your song/track, swipe left to view other categories:
    • artists
    • albums
    • songs
    • et cetera
