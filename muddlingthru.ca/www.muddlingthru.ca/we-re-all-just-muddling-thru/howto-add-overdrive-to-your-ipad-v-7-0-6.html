HowTo: Add OverDrive to Your iPad v 7.0.6 (and borrow an Audiobook)

Friday, April 4, 2014 5:42:00 PM

Prerequisites

  • An email account. This is necessary to create an Apple and Adobe ID
  • An Apple ID. Create one here: https://appleid.apple.com
  • An Adobe ID. Create one here: https://www.adobe.com/ca/account/sign-in.adobedotcom.html
  • An Internet connection.

If you lack an Apple or Adobe ID, it's much easier to create one on a laptop or desktop computer than it is with a tablet, because a tablet lacks a keyboard, which makes it really tricky to enter usernames and passwords.

Steps to Install and Authorize OverDrive

  1. Determine what iPad operating system your using. Settings > General > About > Version. If you're not using 7.0.6, this tutorial might not work for you.
  2. Open the iPad app store.
  3. In the search box on the top right, type overdrive.
  4. In the search results, click overdirve media console.
  5. In the page that opens, click the FREE button that is within the OverDrive Media Console box.
  6. Click the INSTALL button that the FREE button becomes.
  7. Sign in to the iTunes store.
  8. The installation will start and complete in about one minute.
  9. Click the OPEN button that the INSTALL button becomes.
  10. Sign in with your Adobe ID.

Add a Library

  1. From the OverDrive main page, click Add a Library
  2. In the search box, type Salt Spring (two words).
  3. Tap search.
  4. Tap Salt Spring Island Public Library in the search results.
  5. Tap the Star beside the British Columbia Libraries.
  6. The "Source Added" will flash, which means success.
  7. Go "Back" (with the back arrow) three times to get back to OverDrive home.
  8. You have added a library system called British Columbia Libraries.

Borrow a Book

  1. From the OverDrive home page, click British Columbia Libraries.
  2. The library page will open.
  3. Tap on the Audiobook Fiction > All Fiction link.
  4. Click on the Show me... > Available Now link to show only available books.
  5. Tap on any Audiobook (as an experiment, you can choose a specific one later, this is for learning).
  6. The books page will open, and it will be available because of step 4.
  7. Tap Borrow.
  8. Select a library (e.g. Salt Spring Island Public Library) from which to borrow from the drop down list.
  9. Type in your library card number.
  10. Then tap Sign In.

Note: You might receive a message that the book isn't available at your library. We were able to ignore this and to download the book anyway.
