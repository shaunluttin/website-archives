HowTo: Add OverDrive to Your Android (and borrow a Book/Audiobook)

Tags: 2014-117ZP, overdrive, library, android

Friday, June 20, 2014 6:08:10 PM

System requirements: Android v4.0 (or newer). See: http://omc.overdrive.com/v3-available-now/android.php

  1. Upgrade to Android v4.0 if necessary.
  2. Install OverDrive Media Console from the Google Play Store.
  3. Create an AdobeID: https://www.adobe.com/ca/account/sign-in.adobedotcom.html
  4. Open OverDrive > Add a Library (Salt Spring) > Choose Get Books > Open “BC Libraries” > …

Here are more notes on how to do this for an iPad. It’s probably similar after step 4.
