HowTo: Uninstall programs from Windows 7

Wednesday, September 25, 2013 6:36:00 PM

My client has the following list of goals:

  • Uninstall OtShot.
  • Uninstall other unused programs.
  • Remove unused icons from the desktop.

Ultimately, she would like to speed and clean up her computer. We'll talk about that in detail later.

The Basics of Uninstalling from Windows 7

  1. Start
  2. Control Panel
  3. Programs > Uninstall a program
  4. Find the program that you want to uninstall. 
  5. Highlight it.
  6. Click uninstall.
  7. This will be enough to install most well designed programs. You might need to do some or all of the following steps for for poorly designed programs (e.g. OtShot).

Prevent OtShot from Running on Startup

  1. Start > Control Panel > Administrative Tools > System Configuration > Startup
  2. Unselect OtShot application.
  3. Click Apply.
  4. Now OtShot will not run automatically on startup.

Manually Remove OtShot from Program Files

  1. Start > Computer > Acer (C:) > Program Files (x86)
  2. Find the OtShot folder.
  3. Right Click on the Otshot folder.
  4. Choose delete.
  5. Now the OtShot program is no longer in this folder.

Manually Remove OtShot from the Registry

  1. Open regedit
  2. Edit > Find > OtShot
  3. HKEY_CURRENT_USER\Software\OtShot
  4. HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Shared Tools\MSCnofig\startup\OtShot
  5. HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\ZalmanInstaller_otshot
  6. Delete each of these keys and all of there subkeys.
  7. Repeat steps one to two to find other possible instances of registry keys.

Uninstall an Extension from Firefox.

An extension adds functionality to Firefox by allowing a new type of user interaction.

  1. Open Firefox
  2. Click on the orange Firefox bar in the left hand top corner of the window.
  3. Choose Add-Ons
  4. Choose Extensions
  5. Disable or remove any extension that you do not want.
  6. Restart the browser to complete the process.

Disable a Plugin from Firefox.

A plugin adds functionality to firefox without requiring any user interaction.

  1. Follow steps one to three above.
  2. Choose Plugins though instead of Extensions.
  3. Set Plugins that you do not want to use to the Never Active setting.
